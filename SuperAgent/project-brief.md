# rdpproxy v2 -- Описание проекта

## Назначение

Безопасный обратный прокси для протокола RDP (Remote Desktop Protocol) с веб-порталом для SSO-аутентификации через LDAP/Active Directory. Система предоставляет единую точку входа для RDP-подключений с аутентификацией через корпоративный каталог, генерацией .rdp файлов, MITM-проксированием с CredSSP и административной панелью управления.

## Ключевые функции

- **SSO через LDAP/AD**: пользователь авторизуется один раз через веб-портал, система выполняет CredSSP от его имени к целевому RDP-серверу
- **Веб-портал**: логин, список доступных серверов (фильтрация по AD-группам), скачивание .rdp файлов
- **RDP Relay**: MITM-прокси с поддержкой CredSSP/NTLM, MCS patching, Proxy Protocol v2
- **Админ-панель**: управление серверами, шаблонами RDP, пользователями, мониторинг сессий и кластера
- **Плагинная система**: расширяемая архитектура для будущей глубокой работы с RDP (shadow sessions, resource caching, virtual channels)
- **Кластеризация**: подготовка к горизонтальному масштабированию на несколько нод

## Архитектура

Микросервисная, все компоненты в Docker с bridge network. HAProxy на входе мультиплексирует HTTPS и RDP через один порт (8443).
